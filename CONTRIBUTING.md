## QuickStart
Running on linux: `npm run start`<br/>
Running on windows: `npm run startw`

## Branch structure
\<type\>--\<branch name\>

## Git comments
<type>(scope): header
---Blank space---
<Body>
--Blank space---
<Footer>

### types
* feat - feature
* refactor - refactor
* docs - documentation
* ci - continuous integration
* fix - bug fixes

## Formatting Code
Run the following formatting code before pushing your code
- `npm run format`
